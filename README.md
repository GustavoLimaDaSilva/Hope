# **Hope**

### **An English learning app based on React and Typescript.**

Hope is a web app aimed at teaching beginners the fundamentals of the English language using flashcards, AI chat, and fun content.

---

## Development Setup

1. Clone the source locally:

```bash
git clone https://github.com/GustavoLimaDaSilva/Hope.git
```
2. Install the dependencies:
```bash
npm install
```
## To Contribute

1. In your clone, create a feature/refactor branch, e.g.:
```bash
git checkout -b feature/foo
```
2. Run an instance of your branch in a local development environment:
```bash
npm run dev
```
3. Commit your changes:
```bash
git commit -m "feat: added a new foo"
```
4. Push to the branch:
```bash
git push origin feature/foo
```
5. Create a new Pull Request.
